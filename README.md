## 구현 기능
- 로그인/회원가입
- 커뮤니티 글 CRUD, 댓글 CRD
- 게시글 좋아요 등록/삭제
- 게시글 조회수 업데이트
- 지역별, 컨텐츠별 관광지 검색
- 관광지 키워드 검색
- 관광지 즐겨찾기 등록 및 삭제
- 해당 날짜에 방문할 관광지 등록 및 삭제
- 친구 검색, 등록 및 삭제
- 친구가 즐겨찾기 한 관광지 확인
